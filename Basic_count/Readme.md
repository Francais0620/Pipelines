> 对L1PA6-L1HS的保守序列进行突变的统计，看看突变的是哪些地方，以及由什么变成了什么？

### 1. diff-L1PA.v3.py

文件diff-L1PA.v3.py，鉴定了consensus sequence上的突变位点。输入为seq.fata文件,输出为result.txt文件。
